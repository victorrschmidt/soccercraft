Modo de 1 jogador

O objetivo é fazer gol no campo do adversário (metade de cima do campo). Você deve clicar nos comandos para que o jogador que está sendo controlado (jogador com a bola) chegue até a pequena área e chute ao gol.

Modo de 2 jogadores

O objetivo do time azul é fazer o gol no campo de cima, e o objetivo do time vermelho é fazer o gol no campo de baixo. Inicialmente, os jogadores selecionam a posição onde seus jogadores ficarão. Cada time possui um jogador com uma bola, que será o jogador a ser controlado inicialmente por cada time. Cada jogador faz um movimento por turno. Quando um jogador de um dos times chega ao gol para chutar, começa um minijogo, onde o jogador escolhe um lado para chutar, e o jogador que está tentando se defender do gol escolhe um lado para que seu goleiro pule. Se o gol for feito, o jogo termina e o jogador que fez o gol ganha.

Comandos

Mover para cima: O jogador que está com a bola se move para o quadrado de cima do quadrado que ele está. Se houver um jogador no quadrado de cima ou não existir um quadrado acima, o jogo acaba por haver um movimento inválido.

Mover para a direita: O jogador que está com a bola se move para o quadrado à direita do quadrado que ele está. Se houver um jogador no quadrado da direita ou não existir um quadrado na direita, o jogo acaba por haver um movimento inválido.

Mover para baixo: O jogador que está com a bola se move para o quadrado de baixo do quadrado que ele está. Se houver um jogador no quadrado de baixo ou não existir um quadrado abaixo, o jogo acaba por haver um movimento inválido.

Mover para a esquerda: O jogador que está com a bola se move para o quadrado à esquerda do quadrado que ele está. Se houver um jogador no quadrado da esquerda ou não existir um quadrado na esquerda, o jogo acaba por haver um movimento inválido.

Chutar no meio do gol: O jogador que está com a bola chuta no meio do gol. O jogador precisa estar na pequena área adversária para executar esse comando. No modo de 1 jogador, se não houver um goleiro adversário ou o goleiro estiver em outra posição, o gol é feito. Caso contrário, o jogo termina por movimento inválido (não estar na pequena área) ou por defesa do goleiro.

Chutar no canto direito do gol: O jogador que está com a bola chuta no canto direito do gol. O jogador precisa estar na pequena área adversária para executar esse comando. No modo de 1 jogador, se não houver um goleiro adversário ou o goleiro estiver em outra posição, o gol é feito. Caso contrário, o jogo termina por movimento inválido (não estar na pequena área) ou por defesa do goleiro.

Chutar no canto esquerdo do gol: O jogador que está com a bola chuta no canto esquerdo do gol. O jogador precisa estar na pequena área adversária para executar esse comando. No modo de 1 jogador, se não houver um goleiro adversário ou o goleiro estiver em outra posição, o gol é feito. Caso contrário, o jogo termina por movimento inválido (não estar na pequena área) ou por defesa do goleiro.

Passar a bola para cima: O jogador que está com a bola passa ela para um jogador aliado que esteja em uma posição acima dele. É preciso que o espaço entre os dois jogadores esteja vazio, para que a bola não seja interceptada. O jogador que recebeu a bola passa a ser o jogador a ser controlado.

Passar a bola para a direita: O jogador que está com a bola passa ela para um jogador aliado que esteja em uma posição à direita dele. É preciso que o espaço entre os dois jogadores esteja vazio, para que a bola não seja interceptada. O jogador que recebeu a bola passa a ser o jogador a ser controlado.

Passar a bola para baixo: O jogador que está com a bola passa ela para um jogador aliado que esteja em uma posição abaixo dele. É preciso que o espaço entre os dois jogadores esteja vazio, para que a bola não seja interceptada. O jogador que recebeu a bola passa a ser o jogador a ser controlado.

Passar a bola para a esquerda: O jogador que está com a bola passa ela para um jogador aliado que esteja em uma posição à esquerda dele. É preciso que o espaço entre os dois jogadores esteja vazio, para que a bola não seja interceptada. O jogador que recebeu a bola passa a ser o jogador a ser controlado.

Recriar fase: No modo para 1 jogador, este botão recria a fase, com um modelo diferente. No modo para 2 jogadores, este botão reinicia o jogo para o modo de posicionamento dos jogadores.

Trocar de time: Este botão só existe no modo para 1 jogador. Esse botão faz com que o time do jogador mude, mantendo a configuração da fase.

Apagar último movimento: Este botão faz com que o último movimento colocado no painel seja apagado.

Iniciar movimento(s): No modo para 1 jogador, este botão faz com que a lista de movimentos seja executada. No modo para 2 jogadores, este botão faz com que o movimento digitado pelo jogador que está em seu turno seja executado.
